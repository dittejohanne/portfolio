Fidge Buddy

3. semester eksamen - gruppeprojekt

  • Webapplikationskoncept udviklet med HTML, CSS og JavaScript
  • Single Page Application (SPA)
  • Firebase til login og datalagering
  • Moodboard, skitser, wireframes, styletile, mockup
  • Ikoner og logo designet i Adobe Illustrator og Photoshop

Fridgebuddy er en webapp, som hjælper brugeren med at udnytte sine madvare bedre, og derved bidrager til mindre madspil. Webappen holder overblik over brugerens madvarer, og minder om deres udløbsdato ved hjælp af intuitive farver. Ved udløb kan madvaren slettes. Fridgebuddy foreslår også opskrifter, der matcher brugerens indtastede madvare i søgefeltet. Yderligere er en digital indkøbsliste, som vi ønsker at videreudvikle i nærmere fremtid.

Hjemis

1. semester eksamen - individuelt

  • HTML og CSS
  • Moodboard, skitser, wireframes, styletile, mockup
  • Mobile First paradigmet og Responsive Web Design
  • Flatdesign - illustrationer og ikoner udviklet i Adobe Illustrator

Jeg har udviklet en informationsside for Hjemis, som skal fungerer som en sommerkampagne. I mine illustrationer har jeg arbejdet med rene linjer, skarpe kanter og lyse farver.

youSee Musikquiz

2. semester UX/Some - gruppeprojekt

  • Interaktiv User Experience til sponsoraktivering ifm. en sportsbegivenhed
  • Moodboard, skitser, wireframes, styletile, mockup
  • Visuel kommunikation
  • HTML, CSS, Javascript, jQuery og API
  • Agil udviklingsmetode: SCRUM

youSee Musikquiz er en interaktiv brugeroplevelse, der er udviklet til at underholde tilskuere til et sportsevent, før, under og efter eventet. Konceptet omkring musikquizzen er at brugeren skal vælge en musikgenre at quizze i. Herefter skal brugeren begynde at jonglere med fodbolden for at få musikken til at spille, og så gætte det rigtige svar på skærmen.

  • HTML, CSS og JavaScript
  • Moodboard, skitser, wireframes, styletile, mockup
  • Interaktiv audio og video
  • Storytelling

  • Feltresearch: Interviewguide og observationsguide

Kunsthal Aarhus

1. semester - gruppeprojekt

  • HTML, CSS and JavaScript
  • Mobile First paradigmet og Responsive Web Design
  • Interaktiv User Experience

Kunsthal Aarhus er et lille museum i Aarhus, som vi ønskede at tilføre noget mere. Vi oplevede et behov for mere information, men generelt også en bedre fortælling ift. udstillingerne. Derfor udviklede vi en prototype efter mobile first paradigme, som ville kunne bruges når man besøger kunsthallen. Den tilføre en ekstra dimension via. lyd og et interaktivt kort med fortællinger om udstillingerne, når man vandre rundt på museet.
